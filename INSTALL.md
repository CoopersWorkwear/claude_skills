# Installation Guide

## Prerequisites

- Claude Code editor or IDE
- Access to Claude API
- Git for cloning the repository

## Steps

1. Clone this repository:
   ```bash
   git clone https://github.com/CoopersWorkwear/claude_skills.git
   cd claude_skills
   ```

2. Extract the skills (already extracted in this repo)

3. Configure your Claude environment:
   - Copy the skill directories to your Claude plugins folder
   - Update your Claude configuration to include these skills

4. Verify installation:
   - Restart Claude
   - Check that skills are available in the skills menu
