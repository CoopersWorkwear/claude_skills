# Higgsfield Marketplace Cards Skill

## Overview
Create professional, optimized marketplace product cards for various platforms.

## Features

- Product card generation
- SEO optimization
- Category tagging
- Pricing strategy recommendations
- Multi-platform compatibility

## Card Components

- Product title
- Description
- Key features
- Pricing
- Images
- Reviews/ratings
- Call-to-action

## Platforms Supported

- Amazon
- eBay
- Shopify
- Custom storefronts

## Best Practices

1. Use compelling product titles
2. Include key features and benefits
3. Optimize for search
4. High-quality product images
5. Clear pricing and shipping info
