# Claude Skills Repository

A collection of reusable Claude skills for AI-powered marketing and product workflows.

## Skills Included

### 🎨 Higgsfield Generate
AI-powered marketing content generation with multiple modes:
- DTC (Direct-to-Consumer) advertising
- Brand kit customization
- Avatar-based personalization
- Product positioning
- Setup item optimization

**References:**
- Marketing avatars
- Brand guidelines
- Product catalog
- DTC advertising templates
- Troubleshooting guides

### 🛒 Higgsfield Marketplace Cards
Create professional marketplace product cards with:
- SEO optimization
- Product descriptions
- Category optimization
- Pricing strategies

### 📸 Higgsfield Product Photoshoot
Coordinate product photography:
- Photoshoot planning
- Product positioning
- Lighting setup
- Background specifications

### 🎯 Higgsfield Soul ID
Define brand identity:
- Brand essence
- Value proposition
- Target audience alignment
- Brand messaging

## Quick Start

```bash
git clone https://github.com/CoopersWorkwear/claude_skills.git
cd claude_skills
see INSTALL.md for setup
```

## Usage with Claude Code

1. Load the skills from this repository
2. Use skill commands in your Claude Code workflow
3. Reference the documentation in each skill's SKILL.md file
