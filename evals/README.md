# Evaluations

This directory contains evaluation scenarios and test cases for the Claude Skills.

## Purpose

Evaluations ensure skills perform as expected and maintain quality standards.

## Test Scenarios

See `scenarios.md` for detailed test cases covering:
- Content generation quality
- Output format compliance
- Platform-specific requirements
- Edge cases and error handling

## Running Evaluations

1. Load a skill
2. Execute test scenario
3. Review outputs
4. Compare against expected results
5. Document any issues
