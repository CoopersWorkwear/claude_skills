# Contributing to Claude Skills

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-skill`)
3. Make your changes
4. Commit your changes (`git commit -am 'Add new skill'`)
5. Push to the branch (`git push origin feature/your-skill`)
6. Create a Pull Request

## Skill Development Guidelines

- Create a new directory for each skill
- Include a `SKILL.md` file with documentation
- Add reference materials in a `references/` subdirectory
- Follow the existing naming conventions
- Test your skill thoroughly before submitting

## Code Style

- Use clear, descriptive naming
- Comment complex logic
- Keep files organized and maintainable

## Reporting Issues

Use GitHub Issues to report bugs or request features.
